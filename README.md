# esx_vehicleheadlights
## Requirements
- https://github.com/esx-framework/es_extended

## Installation
- CD in your resources/[esx] folder
- Clone the repos
- Add this in your server.cfg : ``start esx_vehicleheadlights``
- Press B inside a vehicle to open the menu
